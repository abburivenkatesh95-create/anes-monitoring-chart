# Anes-Monitoring Chart

A web-based anesthesia monitoring and drug calculator designed for veterinary practice.

## Features

- Real-time anesthesia monitoring chart
- Dual-axis graph with:
  - HR
  - SpO2
  - ETCO2
  - Resp Rate
  - BP (SBP/DBP/MAP)
  - Temperature
  - O2 flow & Iso/Sevo
- Fluid therapy calculator
- Routine anesthesia drug calculator
- Emergency drug calculator (RECOVER guidelines)
- Medication tracking
- Patient case saving (local storage)
- Export / Import cases
- Print-friendly anesthesia record

## How to Run Locally

1. Download or clone the repository
2. Open the folder
3. Open `index.html` in a browser

OR use VS Code Live Server.
